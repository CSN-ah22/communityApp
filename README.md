# 커뮤니티 앱 MVP

이 프로젝트는 커뮤니티 앱을 구현한 MVP입니다. 주요 기능은 회원가입, 로그인, 게시글 목록/상세/작성, 댓글 작성 및 게시물 삭제 기능입니다.

## 기술 스택
- React Native (Expo)
- Firebase

## 주요 기능
- **회원가입/로그인**
- **게시글 작성 및 조회**
- **댓글 작성 및 댓글 수 업데이트**
- **게시물 삭제 기능 (작성자만)**

## 설치 방법
1. `npm install` 명령어로 의존성 설치
2. `npm start`로 Expo 앱 실행
